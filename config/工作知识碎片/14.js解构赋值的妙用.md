# 对象解构赋值的妙用

```javascript
const obj = {
    name: 'zs',
    age: 20,
    hobby: ['唱','跳','rap'],
    ...
}
// 我们需要拿到除hobby以外的属性，应该怎么做?    
const { hobby, ...rest } = obj
console.log(rest)
```

按照以上方式，当我们需要删除一个对象的属性时，也能这么做而不需要使用`delete`关键字或者`Reflect.deleteProterty`
